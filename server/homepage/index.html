CONFESSIONS API

The API is a mechanism which allows queries in the semantic database. It provides access to information from the data catalogue and URIs. These queries can be carried out based on a series of criteria selected by the user and using uniform resource identifiers (URI).

RESPONSE FORMAT

JSON is the default response format

Available References

Data catalogue

Set of calls which allow the API catalogue queries.

Operation Description Example
To obtain all of the data sets /confessions/ http://localhost:3000/confessions/
To obtain data sets into a specific category /confessions/categoryName/ http://localhost:3000/confessions/guilt/
To obtain data sets from a specific city /confessions/id/ http://localhost:3000/confessions/3/
To obtain data sets which match the keyword with any word into the title /confessions/keyword/ http://localhost:3000/confessions/dream/
